# Front-end Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 375px
- Desktop: 1440px

### Primary

### Neutral

--DarkGrayishBlue: hsl(230, 11%, 40%);

- Family: [Barlow](https://fonts.google.com/specimen/Barlow)
